# CodeAlpha Hangman Game

This repository contains a text-based Hangman game developed for the CodeAlpha Python Programming Internship. 

## Features
* Interactive gameplay using Python `while` loops and conditional statements.
* ASCII art that updates dynamically with each incorrect guess.
* A custom dictionary system providing logical and technical hints for the user.
* Input validation to prevent double-guessing or entering invalid characters (like numbers or symbols).
* Built using standard Python libraries, specifically the `random` module.
